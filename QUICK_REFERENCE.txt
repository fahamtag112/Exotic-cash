SERVER MANAGEMENT:
  npm run server                    Basic server start
  npm run server:dev                Start with auto-reload (DEV)
  npm run start                     Production start
  npm run build                     Build TypeScript + React

USING MANAGEMENT SCRIPT:
  ./server-manager.sh start         Start server with diagnostics
  ./server-manager.sh stop          Stop the server
  ./server-manager.sh restart       Restart server
  ./server-manager.sh status        Check server status
  ./server-manager.sh diagnose      Full system diagnostics
  ./server-manager.sh health        Check health endpoint
  ./server-manager.sh reset         Reset database + restart

MONITORING ENDPOINTS:
  curl http://localhost:5000/api/health      Server health + DB status
  curl http://localhost:5000/api/stats       Server uptime + memory
  curl http://localhost:5000/api/pool-status Connection pool statistics

WATCH MONITORING (Real-time):
  watch -n 5 'curl -s http://localhost:5000/api/health | jq .'
  watch -n 5 'curl -s http://localhost:5000/api/stats | jq .'
